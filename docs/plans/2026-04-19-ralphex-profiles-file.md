# План подготовки файла профилей ralphex

1. Подготовить отдельный файл с тремя профилями конфигурации `ralphex`: `cheap`, `balanced`, `strict`.
2. Сохранить файл в `docs/` в удобном для копирования формате с краткими комментариями по назначению каждого профиля.
3. Проверить содержимое файла и рабочее дерево, чтобы убедиться, что профили сохранены корректно.
